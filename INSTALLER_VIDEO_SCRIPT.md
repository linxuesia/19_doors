## 三、量尺任务 (1:00 - 2:15)

### 3.1 查看任务信息
**操作**:
1. 点击一个量尺任务卡片 → 进入详情
2. 展示任务信息卡片：地址、客户姓名电话、房屋面积
3. 向下滑动到「量尺记录」区域

### 3.2 拍照记录
**操作**:
1. 点击「添加量尺照片」→ 选择相册或拍照
2. 拍摄一张现场照片 → 照片出现在记录列表中
3. 点击照片下方的「点击添加说明」→ 输入文字说明
4. 也可以点快捷标签快速填写：「窗户整体尺寸」「墙体结构情况」等
5. 再添加第二张照片，填写说明
6. 展示照片可以点开大图预览

### 3.3 提交量尺
**操作**:
1. 底部展示两个按钮：「拍照记录」和「提交量尺完成」
2. 点击「提交量尺完成」→ 弹窗确认 → 提交成功
3. 状态变为「已量尺」，自动返回任务列表

**解说**: "量尺详情页展示客户信息和房屋面积。拍照记录支持相册选取和现场拍摄，每张照片配文字说明。快捷标签一键填入常用描述，提高效率。全部记录完成后一键提交，任务状态自动更新为已量尺。"

## 四、施工任务 (2:15 - 3:30)

### 4.1 查看工单信息
**操作**:
1. 切换到「施工」tab → 点击一个「施工中」的工单
2. 展示工单信息：地址、客户姓名电话、安装时间
3. 展示「量尺参考」区域 → 点击展开
4. 查看量尺阶段拍摄的照片和说明，辅助施工

### 4.2 更新施工进度
**操作**:
1. 向下滑动到「进度更新与图片」区域
2. 阶段选择器展示当前阶段（如「材料入场」）→ 点击可切换阶段
3. 展示预设的阶段列表：材料入场、旧窗拆除、窗框安装、玻璃安装、打胶密封、收尾清洁
4. 点击「添加图片」或「拍照上传」→ 拍摄施工照片
5. 添加完成后点击底部「进度更新与图片」→ 弹窗确认 → 提交成功

### 4.3 申请验收
**操作**:
1. 回到工单详情（施工中状态）
2. 底部展示「申请验收」按钮
3. 点击 → 弹窗「提交验收申请后需等待管理员审核」→ 确认
4. 状态变为「待验收」

**解说**: "施工详情首先展示工单基本信息。量尺参考区域展示量尺阶段拍摄的现场照片，安装工可以对照参考，精准施工。进度更新时选择当前阶段，上传施工照片，方便门店和客户实时了解进展。完工后点击申请验收，管理员审核通过即可完成订单。"
